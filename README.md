# Click Creator Cleaner

Интерактивное приложение на C# Windows Forms для создания и удаления элементов управления.

## Функциональность

### 🖱️ Создание (левая кнопка мыши)
- Первый клик → кнопка ("Кнопка 1")
- Второй клик → текстовое поле ("Текст 2")
- Третий клик → кнопка ("Кнопка 3")
- И так далее (чередование)
- Новый элемент создаётся в месте клика
- Автоматическая нумерация

### 🗑️ Удаление (правая кнопка мыши)
- Удаляет все созданные кнопки и текстовые поля
- Счётчик элементов обнуляется

## Интерфейс

- Форма — основная область для кликов
- Клик левой кнопкой → создание
- Клик правой кнопкой → удаление всех

## Пример работы

1. Клик левой в (100,100) → кнопка "Кнопка 1"
2. Клик левой в (200,200) → поле "Текст 2"
3. Клик левой в (300,300) → кнопка "Кнопка 3"
4. Клик правой → все созданные элементы удалены

## Куда можно встроить

- В визуальные редакторы
- В демонстрацию создания элементов
- В примеры обработки событий мыши
- В обучающие проекты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Исходные элементы формы (если они есть) не удаляются. Удаляются только созданные программой кнопки и текстовые поля.
